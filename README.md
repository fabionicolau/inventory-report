<h1> Sobre </h1>

<p> Projeto Inventory Report onde o desafio era implementar um gerador de relatórios que recebe como entrada arquivos com dados de um estoque e gera, como saída, um relatório acerca destes dados. </p>
<p> Além disso, os dados são obtidos através de arquivos CSV, JSON ou XML e deveriam ser desenvolvidas funcionalidades para leitura desses arquivos </p>
